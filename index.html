Help

Getting Started

1. Register professor (NFC card or password)
2. Install the app for offline use (see below)
3. Open the installed app from home screen
4. Upload a student list file
5. Press Attend NFC or Attend Code to start

After installing, you do not need to visit this website again. All data is stored locally on your device.

Install for Offline Use

iPhone / iPad: Open this page in Safari, tap the Share button (square with arrow), then choose "Add to Home Screen".

Android: Open menu ☰ and choose "Install App (PWA)", or tap Chrome menu (...) > "Add to Home screen".

Student List File Format

Plain text file (.txt) with:

Line 1: Course Name
Line 2: Professor Name
Line 3: Student Name,last4digits (or tab)
Line 4: Student Name,last4digits (or tab)
...

Last 4 digits of student number are used for Attend Code verification. Name and digits can be separated by comma or tab.

Example:

AF1234 Business Analytics
Professor Mary Chen
Peter Adoby,1234
Peter Bai,5432
David Lim,9876

Modes

Attend NFC — Students tap NFC cards continuously
Attend Code — Scan QR codes for attendance
Find — Scan NFC card to locate a student

Student Actions

Manual — Manual attendance (double confirm + professor NFC)
Log — View attendance history
Re-reg — Re-register NFC/QR (professor NFC required)

Multiple Courses

The app supports multiple courses. You can upload multiple student lists. Switch between courses from the Courses section in the menu.

Data Management

You can export the attendance record or transfer the records via Bluetooth. You can backup and restore app local storage to another device.
